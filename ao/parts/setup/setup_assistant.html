Welcome

This app uses the StayPrivate secure video, audio and screen sharing plug-in to ensure that you get the best performance possible for your internet connection.
To get started, please install the plug-in by clicking the button below.

  • Software Check
  • Connection Check

Please follow the instructions below.

Step 1: Click the AOSecureVideo.exe file in the bottom left of the browser window. Click 'Save File' in the popup. In the popup window or popup bar below, click 'Run'. Click the AOSecureVideo.dmg file in the bottom left of the browser window. Double-click the AOSecureVideo.dmg file from the Downloads popup.

Step 2: The installer will run. Click 'Finish' when it is done. Click 'Run' to install the plugin. Double-click the AOSecureVideo.exe file in the 'Downloads' popup. Click the 'Finder' icon to open installation window. Double-click the installer icon.

Step 3: Please wait, this screen will automatically update. The installer will run, click 'Finish' when it is done. When the installation has finished, click 'OK' and return to the browser.
