Import Contacts Help

Importing your existing contact lists is the safest and easiest way to add contacts in Quorum. Imported lists are added to your own private list, which is accessible to you only. To create Quorum contacts or groups from your private contact list, select "NEW", and follow the on-screen instructions.

This two-step process has been designed to enable you to confidently upload existing contact lists without needing to check them or worrying about contacting people accidentally. Once you have uploaded your lists, you can then easily search and select the contacts with whom you wish to communicate.

Exporting your contact lists is usually a simple process. Click on the examples below to see how to export from the following environments:

  • Exporting from Outlook 2010
  • Exporting from a Mac
  • Exporting from Gmail
  • Exporting from a Microsoft account (Outlook.com, Live, Hotmail, MSN)

Exporting from Outlook 2010

  1. In the Outlook menu, select File, Options.
  2. In the Outlook Options window, select Advanced.
  3. Select Export.
  4. In the Import and Export Wizard, select Export to a File.
  5. Select Comma Separated Values (Windows).
  6. Choose the contacts you wish to export.
  7. Once you have created the file, simply drag it into the box on this screen (you need to exit help first).

Exporting from a Mac

  1. Open the Applications Menu.
  2. Select Contacts.
  3. Select the contact list you would like to export, e.g. All Contacts.
  4. From the Contacts menu, Select File, Export, then Export Vcard... .
  5. Once you have created the file, simply drag it into the box on this screen (you need to exit help first).

Exporting from Gmail

  1. From Gmail, click Gmail at the top-left corner of your Gmail page.
  2. Select Contacts.
  3. Select More...
  4. Select Export.
  5. Choose the contacts you wish to export and select Outlook CSV format.
  6. Once you have created the file, simply drag it into the box on this screen (you need to exit help first).

Exporting from a Microsoft account (Outlook.com, Live, Hotmail, MSN)

  1. Go to the People section of your Microsoft account (from your email account, you can find this via the icon top left).
  2. Select Manage.
  3. Select More...
  4. Select Export.
  5. Choose which contacts to export and select Outlook CSV format.
  6. Once you have created the file, simply drag it into the box on this screen (you need to exit help first).
