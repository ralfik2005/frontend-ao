1. Find your PC’s audio settings

  1. Click Start and type “sound” into the search bar. A list of search matches is displayed.
  2. In the search results, select Sound under the Control Panel list.

The Sound screen is displayed.

2. Make sure that your PC is using the correct headphones or speakers

  1. In the Sound screen, select the Playback tab and click on the device the system uses. If none is selected, choose a default by clicking Set Default.

  2. Click Properties.
  3. In the Properties screen, make sure Use this device (enable) is selected in the Device usage drop-down list.

  4. Select the Levels tab.
  5. Make sure that the Speakers/Headphones volume is high enough or is not muted and click OK.

3. Make sure your computer is using the correct microphone

  1. In the Sound screen, select the Recording tab.
  2. Click on the recording device that the system uses. It should have a tick next to it. If it does not, set it as your default by clicking Set Default.

  3. Click Properties.
  4. In the Properties screen, make sure that the Use this device (enable) option is selected in the Device usage drop-down list.

  5. Select the Levels tab.
  6. Make sure that the Microphone volume is set to maximum and is not muted.
  7. Check that the Microphone Boost set to minimum to disable it and click OK.

  8. If you are having problems with the microphone, click Start > All Programs > Accessories > Sound Recorder to display the Sound Recorder:

    Try to record your voice through your microphone, then play it back.

    If you cannot hear what you recorded, re-display the Sound screen (see above) and play with the microphone settings until the recording test works.

    If you can hear your recording, make sure that your audio settings are correct.

4. Make sure your audio settings are correct

  1. Sign in.
  2. Click Settings > Audio/Video Setup.

  3. In the Audio/Video settings dialog, select the devices you chose in your PC’s Sound screen from the Microphone and Speakers drop-down lists and click OK.
