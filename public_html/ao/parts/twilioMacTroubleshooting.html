Access the Sound Preferences on your Mac

  1. Click Apple Menu > System Preferencesâ€¦.

  2. In the System Preferences window, select Sound.
  3. The Sound preferences window appears.

  4. In the Sound preferences window, select Output. This displays the Output preferences.


  5. In the Select a device for sound output field, the sound output device is selected. If you see several devices in this list, select the one you prefer for sound output. The selection shown here is the default output for sound on your Mac.
  6. Use the slider to adjust the Balance for this device. For an even sound, it is best to set this in the middle.
  7. Use the slider to adjust the Output volume. When using the system, ensure Mute is not selected at the foot of the display. You can adjust the volume setting using the speaker keys on some Mac keyboards.
